🔑 Chave do Google Places

Cole sua chave (AIza…). Fica salva só neste aparelho. Para funcionar, no Google Cloud precisa:

  1. Ativar Maps JavaScript API
  2. Ativar Places API
  3. Restringir a chave por Referenciador HTTP ao endereço deste site
Não tem chave? Abrir credenciais do Google Maps. O Google dá ~US$ 200/mês de crédito grátis.
